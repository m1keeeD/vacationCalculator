# vacationCalculator
# Spring Boot application for calculating vacation pay 
    Assignment from the course

## Used techologies:

        - Java Core
        - spring Boot
        - HTTP API
            'GET /calculate'

## App logic

    Based on average employee's salary from 12 months and the number of vacation days.
    Returns the sum employee need to be paid
